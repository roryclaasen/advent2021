# Usage

<!-- usage -->
```sh-session
$ npm install -g @roryclaasen/advent2021
$ advent2021 COMMAND
running command...
$ advent2021 (-v|--version|version)
@roryclaasen/advent2021/1.6.0 linux-x64 node-v16.13.0
$ advent2021 --help [COMMAND]
USAGE
  $ advent2021 COMMAND
...
```

## `advent2021 Day1`

```
USAGE
  $ advent2021 Day1

OPTIONS
  -p, --part=1|2  Part to run
  -t, --test      Run using test data

ALIASES
  $ advent2021 day1
  $ advent2021 day:1
```
