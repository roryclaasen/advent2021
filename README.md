## `advent2021 help [COMMAND]`

display help for advent2021

```
USAGE
  $ advent2021 help [COMMAND] [--all]

ARGUMENTS
  COMMAND  command to show help for

FLAGS
  --all  see all commands in CLI

DESCRIPTION
  display help for advent2021
```
