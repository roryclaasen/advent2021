# Usage

<!-- usage -->
```sh-session
$ npm install -g @roryclaasen/advent2021
$ advent2021 COMMAND
running command...
$ advent2021 (--version)
@roryclaasen/advent2021/2.10.0 win32-x64 node-v14.17.0
$ advent2021 --help [COMMAND]
USAGE
  $ advent2021 COMMAND
...
```

## `advent2021 day1`

```
USAGE
  $ advent2021 day1 [-p 1|2] [-t]

FLAGS
  -p, --part=<option>  Part to run
                       <options: 1|2>
  -t, --test           Run using test data

ALIASES
  $ advent2021 day:1
```

```
FLAGS
```

```
DESCRIPTION

```
